Comandos para agregar una dirección IP en una interfaz de red del router:
	int(interface) nombre_interfaz
	desc(description) 'mensaje o descripción sobre la conexión'
	ip(interface_protocolo) add(agregate) direción_IP máscara_subred
	no shut(no_apagar)
	exit(salir) 
	
Ejemplo:
	int g0/0(gigabit_ethernet)
	desc Interfaz de red del LAN router
	ip add 10.0.0.1 255.255.255.0
	no shut 
	exit

Comandos para implementar el enrutamiento estático en routers cisco en el modo de configuración global:
	Opción_1
		ip route dirección_IP_red_destino mascara_subred_destino dirección_IP_interfaz_router_vecino

	Opción_2
		ip route dirección_IP_red_destino mascara_subred_destino dirección_interfaz_router_local

Ejemplo:
	Opción_1
		ip route 172.0.100.0 255.255.192.0 100.0.5.2

	Opción_2
		ip route 172.0.100.0 255.255.192.0 g0/1

Configurando direccionamiento IP a los routers

	Configuración modelo 
		en
		conf t
		int nombre_interfaz
		desc 'mensaje o descripción sobre la conexión'
		ip add direción_IP máscara_subred
		no shut
		exit 
		ip route IP_red_Dapiato_Azua mascara_subred_Dapiato_Azua IP_interfaz_Claro
		end
		copy run start
		reload
