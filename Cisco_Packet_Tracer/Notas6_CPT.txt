CISCO IOS: nombre del sistema operativo.
Nodos:
	1. usuario
		simbolo-representación: >
		comando N/A
	2. privilegiado
		simbolo-representación: #
		comando: enable
		abreviatura: en
	3. configuracion global
		simbolo-representación: (config)# 
		comando: configure: terminal
		abreviatura: conf t
	4. configuracion de interfaz
		simbolo-representación: (config-if)#
		comando: interface nombre-interfaz
		abrerviatura: int nombre-abreviado
	5. configuracion de linea
		simbolo-representación: (config-line)
		comando: line nombre-linea id-linea
		abrebiaruta: li nombre-abreviado id-linea

Nota 1. Cada modo de operacion, tiene un grupo de  comando que solo se pueden ejecutar dentro de dicho nodo. 

Nota 2. Para salir de un modo y volver al modo anterior , usamos el comando "exit", y para salir de un modo e ir directamente al modo privilegiado, usamos el comando "end".

		.224
	5. Vuelvo a ensamblar la nueva máscara de subred.

NOTAS 5


Comandos para agregar una dirección IP en una interfaz de red del router:
	int(interface) nombre_interfaz
	desc(description) 'mensaje o descripción sobre la conexión'
	ip(interface_protocolo) add(agregate) direción_IP máscara_subred
	no shut(no_apagar)
	exit(salir) 
	
Ejemplo:
	int g0/0(gigabit_ethernet)
	desc Interfaz de red del LAN router
	ip add 10.0.0.1 255.255.255.0
	no shut 
	exit

Comandos para implementar el enrutamiento estático en routers cisco en el modo de configuración global:
	Opción_1
		ip route dirección_IP_red_destino mascara_subred_destino dirección_IP_interfaz_router_vecino

	Opción_2
		ip route dirección_IP_red_destino mascara_subred_destino dirección_interfaz_router_local

Ejemplo:
	Opción_1
		ip route 172.0.100.0 255.255.192.0 100.0.5.2

	Opción_2
		ip route 172.0.100.0 255.255.192.0 g0/1

		exit

		end
